# 하향 타입 변환을 캡슐화 

###### Encapsulate Downcast 

메서드가 반환하는 객체를 호출 부분에서 하향 탑입 변환해야 할 땐
하향 타입 변환 기능을 메서드 안으로 옮기자

</br>

[보기]

- 변경 전

  ```java
  Object lastReading() {
      return readings.lastElement();
  }
  ```

</br>

- 변경 후

  ```java
  Reading lastReading() {
      return (Reading)readings.lastElement();
  }
  ```

</br>


[방법]

1. 메서드 호출의 결과를 반환된 값을 하향 타입 변환해야 하는 각종 상황을 찾자
   (컬렉션이나 이터레이터를 반환하는 메서드에서 자주 볼 수 있음)

</br>

2. 하향 타입 변환 코드를 그 메서드 안으로 옮기자
   (컬렉션을 반환하는 메서드에 컬렉션 캡슐화를 적용하자)
