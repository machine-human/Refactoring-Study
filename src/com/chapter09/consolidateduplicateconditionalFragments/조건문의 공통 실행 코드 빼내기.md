# 조건문의 공통 실행 코드 빼내기

###### Consolidate Duplicate Conditional Fragments

조건의 모든 절에 같은 실행 코드가 있을 땐 같은 부분을 조건문 밖으로 빼자

</br>

[보기]

- 변경 전

  ```java
  if (isSpecialDeal()) {
      total = price * 0.95;
  	send();
  } else {
  	total = price * 0.98;
  	send();
  }
  ```

- 변경 후

  ```java
  if (isSpecialDeal()) {
      total = price * 0.95;
  } else {
  	total = price * 0.98;
  }
  send();
  ```

</br>


[방법]

1. 조건에 상관없이 공통적으로 실행되는 코드를 찾자

</br>

2. 공통 코드가 조건문의 앞에 절에 있을 땐 조건문 앞으로 빼자

</br>

3. 공통 코드가 조건문의 끝 절에 있을 땐 조건문 뒤로 빼자

</br>

 	4. 공통 코드가 조건문의 중간 절에 있을 땐 앞 뒤의 코드와 위치를 바꿔도 되는지 판단
     (바꿔도 된다면 앞의 단계처럼 조건문의 앞이나 뒤로 빼자)

</br>

5. 공통 코드 명령이 둘 이상일 땐 메서드로 만들자

</br>

[저자 왈]

- 예외 처리에도 이 방식을 적용할 수 있음
- 코드가 try 구간과 모든 catch 구간 안의 예외 발생 명령 뒤의 공통적으로 들어 있으면,
- 그 코드를 final 구간으로 옮기면 됨
