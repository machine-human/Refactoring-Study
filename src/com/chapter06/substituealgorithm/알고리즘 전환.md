# 알고리즘 전환 - Substitute Algorithm

알고리즘을 더 분명한 것으로 교체해야 할 땐 해당 메서드의 내용을 새 알고리즘으로 바꾸자
</br>

[보기]

- 수정 전

  ```java
  private String foundPerson(String[] people) {
  	for (int i = 0; i < people.length; i++) {
  		if (people[i].equals("Don")) {
  			return "Don";
  		}
  		if (people[i].equals("John")) {
  			return "John";
  		}
  
  		if (people[i].equals("Kent")) {
  			return "Kent";
  		}
  	}
  	return "";
  }
  ```
</br>

- 수정 후

  ```java
  private String foundPerson(String[] people) {
      List candidates = Arrays.asList(new String[] {"Don", "John", "Kent"});
  	for (int i = 0; i < people.length; i++) {
          if(candidates.contains(people[i])) {
  			return people[i];
          }
      }
      return "";
  }
  ```
</br>

[방법]

1. 교체할 간결한 알고리즘을 준비 후 컴파일을 실시

2. 새 알고리즘을 실행하면서 여러 번의 테스트를 실시

   (모든 테스트 결과가 같으면 성공)

3. 결과가 다르게 나온다면, 기존 알고리즘 테스트와 디버깅을 실시해 비교
