# 메서드 내용 직접 삽입

###### Inline Method

메서드 기능이 너무 단순해서 메서드명만 봐도 너무 뻔할 땐 그 메서드의 기능을 호출하는 메서드에 넣어버리고 그 메서드는 삭제하자



[보기]

- 변경 전

  ```java
  int getRating() {
  	return (moreThanFiveLateDeliveries()) ? 2 : 1;
  }
  
  boolean moreThanFiveLateDeliveries() {
  	return numberOfLateDeliveries > 5;
  }
  ```



- 변경 후

  ```java
  void getRating(double amount) {    
      return (numberOfLateDeliveries > 5) ? 2 : 1;
  }
  ```



[방법]

1. 메서드가 재정의되어 있지 않은지 확인
   (그 메서드가 하위 클래스에 재정되어 있다면 메서드 내용 직접 삽입을 실시하지 말아야 함.
   없어진 메서드를 재정의하는 일이 생겨서는 안 되므로)
2. 그 메서드를 호출하는 부분을 모두 찾고
3. 각 호출 부분을 메서드 내용으로 교체
4. 테스트 실시
5. 메서드 정의 삭제
